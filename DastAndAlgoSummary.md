## *Graph Algorithms*
A graph with V vertices and E edges
| *Algorithm*     | *Time complexity*               |
| -----------     | -----------                     |
| CC              | *O(\|V\| + \|E\|)*              |
| DFS             | *O(\|V\| + \|E\|)*              |
| BFS             | *O(\|V\| + \|E\|)*              |

### *Array Algorithms*
An array with n elements
| *Algorithm*     | *Time complexity*      |
| -----------     | -----------            |
| quickSort       | *O(n\*log(n))* (mean)  |
| mergeSort       | *O(n\*log(n))*         |
| bubbleSort      | *O(n^2)*         |
