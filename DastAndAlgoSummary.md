## *Graph Algorithms*
A graph with V vertices and E edges
| *Algorithm*     | *Time complexity*               |
| -----------     | -----------                     |
| CC              | *O(\|V\| + \|E\|)*              |
| DFS             | *O(\|V\| + \|E\|)*              |
| BFS             | *O(\|V\| + \|E\|)*              |

### *Array Algorithms*
An array with n elements
| *Algorithm*     | *Time complexity*      | *Time complexity - Worst Case* |
| -----------     | -----------            | -----------                    |                                 
| quickSort       | *O(n\*log(n))* (mean)  | *O(n^2)*                         |
| mergeSort       | *O(n\*log(n))*         |                                |
| bubbleSort      | *O(n^2)*               |                                |
| quickSelect     | *O(n)*  (mean)         | *O(n^2)*                         |
